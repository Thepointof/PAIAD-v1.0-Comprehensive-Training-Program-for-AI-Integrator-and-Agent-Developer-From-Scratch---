---

## 📖 О проекте

Это не просто промпт, а **структурированный обучающий комплекс**. В мире, где технологии меняются ежемесячно, сложно найти актуальный путь. Этот проект решает проблему хаотичного обучения, выступая в роли **персонального методиста**, который координирует работу узкопрофильных «учителей».

### 💡 Основная идея
Создать экспериментальный инструмент, который помогает не только писать код, но и понимать:
*   Архитектуру ИИ-систем.
*   Бизнес-интеграцию.
*   Сложные взаимодействия в роях агентов.

---

## ⚡ Почему это работает?

1.  **Модульность для экономии контекста**  
    Вместо одного гигантского промпта система генерирует специализированные промпты для каждого этапа. Вы работаете с «Учителем» в отдельном чате, а «Методист» хранит общую карту прогресса.
2.  **Фокус на методах и инструментах (Новинка v1.2)**  
    Перед каждым практическим заданием учитель обязан выдать список ключевых методов/функций. Больше никакой магии «я не знаю, чем это решать».
3.  **Педагогические принципы**  
    *   🧠 **Техника Фейнмана:** Объяснения простым языком с бытовыми аналогиями.
    *   🔢 **Правило трёх единиц:** Не больше 3 новых фактов за блок.
    *   🛠 **Интерактивность:** Тесты, задачи, код пишете вы (учитель только направляет).
4.  **Честный трекинг**  
    Система отчётов позволяет фиксировать прогресс. Это создаёт «карту знаний», к которой можно вернуться для повторения.

## 🏗 Роли в системе

| Роль | Описание |
| :--- | :--- |
| 🤖 **Методист** | Центральный узел. Управляет прогрессом, анализирует отчёты, рекомендует шаги. |

| 🌐 **Учитель API** | HTTP, LLM API, российские сервисы (GigaChat, YandexGPT). |
| 📚 **Учитель RAG** | Векторные базы, поиск, оценка качества. |
| 🤖 **Учитель Agents** | LangChain, LangGraph, память, планирование, бизнес-интеграция. |
| 🐝 **Исследователь Роев** | Продвинутый блок. Эксперименты, архитектура мультиагентных систем. |

## 🚀 Быстрый старт

1.  Скопируйте промпт **Методиста** в чат с LLM.
2.  Дождитесь приветственного сообщения.
3.  Используйте команды для навигации.

### 📋 Список команд

| Команда | Описание |
| :--- | :--- |
| `Шаг1` | Активировать **Учителя Python** (Фундамент) |
| `Шаг2` | Активировать **Учителя API** (Интеграции) |
| `Шаг3` | Активировать **Учителя RAG** (Базы знаний) |
| `Шаг4` | Активировать **Учителя Agents** (Разработка) |
| `Шаг5` | Активировать **Исследователя Роев** (Experimental) |
| `карта` | Показать учебный план с отметками о пройденном `[V]` |
| `повторить` | Список пройденных тем для закрепления |
| `help` | Справка по командам |

## ⚠️ Важные особенности использования

> [!WARNING]
> **Ручной перенос отчётов**  
> Поскольку это промпт, а не полноценное приложение, система не имеет памяти между сессиями автоматически.
> 1. После завершения темы **Учитель генерирует Отчёт**.
> 2. Вы **копируете его** и отправляете **Методисту**.
> 3. Это обновляет «Карту прогресса». Без отчёта Методист не знает, что вы прошли тему.

> [!TIP]
> **Разделение чатов**  
> Для экономии контекста рекомендуется запускать промпт Учителя (который выдаст Методист) в **новой вкладке/чате**. В основном чате остаётся только Методист.
